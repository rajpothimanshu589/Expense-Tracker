# Expense Tracker Application

The Expense Tracker is a full-stack web application that helps users manage their daily expenses efficiently.  
Users can add, view, update, and delete expenses, allowing them to track spending habits and maintain better financial control.

## Features
- Add new expenses with amount, category, and date
- View expense list with details
- Update and delete expenses
- Categorized expense tracking
- Responsive and user-friendly UI
- Secure backend APIs
- Error handling and validations

## Tech Stack

### Frontend
- React.js
- HTML5
- Tailwind CSS
- Shadcn/ui
- JavaScript (ES6+)

### Backend
- Node.js
- Express.js

### Database
- MongoDB
